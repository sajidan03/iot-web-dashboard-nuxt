# IoT Monitoring & Controlling Dashboard

Selamat datang di **IoT Monitoring & Controlling Dashboard**, aplikasi berbasis **Nuxt.js** untuk memantau dan mengontrol perangkat IoT secara real-time.  

Proyek ini memungkinkan pengguna untuk:  
- Memantau sensor (misal suhu, kelembaban, jarak) secara live  
- Mengontrol aktuator (misal relay, motor, servo) dari web  
- Menyimpan dan menampilkan data historis
